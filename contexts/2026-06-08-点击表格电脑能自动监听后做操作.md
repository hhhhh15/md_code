# 点击飞书表格，将key传输到Android studio中全局搜索

日期：2026/06/08

## 目标

点飞书某行 → 自动把key填到Android Studio的全局搜索框里。

## 方案

### 方案一：飞书自动化 + 脚本监听（推荐）
原理：飞书多维表格有「自动化」功能，可以在点击按钮/行变化时触发 Webhook，本地跑一个小脚本收到后自动操作 Android Studio。
但问题是：飞书自动化触发的是「字段变化」或「按钮点击」，不是"选中行"这个动作，所以需要在表格里加一个按钮列。
流程：
飞书表格点按钮 → 触发Webhook → 本地Python脚本收到key值 
→ 用pyautogui模拟键盘 → 激活AS窗口 → Shift+Ctrl+F → 粘贴key → 回车

### 方案二：Chrome插件（更直接）
写一个浏览器插件，监听飞书表格页面，当你点击某行时，读取第四列的值，然后通过本地Native Messaging协议发给本地脚本操作AS。
实现难度比方案一高一些。

### 方案三：最简单粗暴——快捷键脚本
不改飞书，在飞书里手动复制key后，按一个全局快捷键，脚本自动：

激活Android Studio窗口
打开全局搜索 Shift+Ctrl+F
粘贴剪贴板内容

**核心**
Windows API 来操作窗口

#### 使用流程
1.pip安装keyboard安装包：pip install keyboard pyautogui pyperclip pywin32
2.as_search.py
3.cd C:\Users\homerun\.claude\contexts\a_daiMa
4.python as_search.py
5. Ctrl+C复制key
4.你按 Ctrl+Alt+F
    ↓
keyboard 库检测到这个按键组合
    ↓
调用 trigger_search()
    ↓
win32gui 扫描所有窗口找到AS的hwnd
    ↓
SetForegroundWindow(hwnd) → AS窗口跳到最前面
    ↓
pyautogui 模拟键盘按 Ctrl+Shift+F → AS内打开全局搜索
    ↓
pyautogui.typewrite(key) → 把剪贴板的key打进搜索框
